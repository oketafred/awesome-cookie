# Awesome Cookie

This is a simple Javascript API that makes working and manipulating cookies a breeze.

## Installation
```
npm install awesome-cookie
```
## Usage

```javascript
import { setCookie, getCookie, removeCookie } from 'awesome-cookie';

setCookie('name', 'value');

getCookie('name');

removeCookie('name');
```
